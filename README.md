# Web Forum Platform

Full-stack web forum app with:

- ✅ Next.js (frontend)
- ✅ Express.js + MongoDB (backend)
- ✅ JWT Auth
- ✅ Real-time chat with Socket.io
- ✅ MongoDB Atlas integration

## How to Run

### Server

```bash
cd server
npm install
npm run dev
```

### Client

```bash
cd client
npm install
npm run dev
```

## .env Config Example

In `server/.env`:

```env
MONGO_URI=your_mongodb_atlas_uri
JWT_SECRET=your_secret
```
